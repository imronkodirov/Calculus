Calculus Curve Sketching

Selected Exercises: 1, 9, 13, 15, and 21

Question 1: \( y = x^3 + 3x^2 \)

1. Domain

Since it is a polynomial, the domain is \( (-\infty, \infty) \).

2. Asymptotes

Vertical: None
Horizontal: None

3. Critical Points

First derivative: \( y' = 3x^2 + 6x = 3x(x + 2) \)

Setting \( y' = 0 \), we get \( x = 0 \) and \( x = -2 \).

Plugging these back into the original equation gives the points:
(0, 0) and (-2, 4).

4. Inflection Points

Second derivative: \( y'' = 6x + 6 = 6(x + 1) \)

Setting \( y'' = 0 \), we get \( x = -1 \).

Plugging \( x = -1 \) into the original equation gives the point:
(-1, 2).

Question 9: \( y = \frac{x}{x - 1} \)

1. Domain

The denominator cannot be zero. Domain: \( (-\infty, 1) \cup (1, \infty) \).

2. Asymptotes

Vertical: \( x = 1 \) (denominator is 0, numerator is not)
Horizontal: \( y = 1 \) (since \( \lim_{x \to \pm\infty} \frac{x}{x-1} = 1 \))

3. Critical Points

First derivative (Quotient Rule):
\( y' = \frac{(1)(x-1) - (x)(1)}{(x-1)^2} = \frac{-1}{(x-1)^2} \)

Since the numerator is never 0, \( y' \neq 0 \). None.

4. Inflection Points

Second derivative:
\( y'' = \frac{d}{dx}[-(x-1)^{-2}] = 2(x-1)^{-3} = \frac{2}{(x-1)^3} \)

Since the numerator is never 0, \( y'' \neq 0 \). None.

Question 13: \( y = \frac{x}{x^2 - 4} \)

1. Domain

Denominator \( x^2 - 4 \neq 0 \Rightarrow x \neq \pm 2 \).
Domain: \( (-\infty, -2) \cup (-2, 2) \cup (2, \infty) \).

2. Asymptotes

Vertical: \( x = -2 \) and \( x = 2 \)
Horizontal: \( y = 0 \) (degree of denominator > degree of numerator)

3. Critical Points

First derivative:
\( y' = \frac{(1)(x^2 - 4) - x(2x)}{(x^2 - 4)^2} = \frac{-(x^2 + 4)}{(x^2 - 4)^2} \)

Setting \( y' = 0 \Rightarrow -(x^2 + 4) = 0 \). No real solutions. None.

4. Inflection Points

Second derivative:
\( y'' = \frac{2x(x^2 + 12)}{(x^2 - 4)^3} \)

Setting \( y'' = 0 \Rightarrow x = 0 \).
Point: (0, 0).

Question 15: \( y = \frac{x^2}{x^2 + 3} \)

1. Domain

\( x^2 + 3 \) is always positive. Domain: \( (-\infty, \infty) \).

2. Asymptotes

Vertical: None
Horizontal: \( y = 1 \) (since \( \lim_{x \to \pm\infty} \frac{x^2}{x^2+3} = 1 \))

3. Critical Points

First derivative:
\( y' = \frac{2x(x^2 + 3) - x^2(2x)}{(x^2 + 3)^2} = \frac{6x}{(x^2 + 3)^2} \)

Setting \( y' = 0 \Rightarrow 6x = 0 \Rightarrow x = 0 \).
Point: (0, 0) (Local minimum).

4. Inflection Points

Second derivative:
\( y'' = \frac{18(1 - x^2)}{(x^2 + 3)^3} \)

Setting \( y'' = 0 \Rightarrow 1 - x^2 = 0 \Rightarrow x = \pm 1 \).

Points: (1, 1/4) and (-1, 1/4).

Question 21: \( y = (x - 3)\sqrt{x} \)

1. Domain

Due to the square root, \( x \ge 0 \). Domain: \( [0, \infty) \).

2. Asymptotes

Vertical: None
Horizontal: None (as \( x \to \infty, y \to \infty \))

3. Critical Points

Rewrite as: \( y = x^{3/2} - 3x^{1/2} \)
First derivative: \( y' = \frac{3}{2}x^{1/2} - \frac{3}{2}x^{-1/2} = \frac{3(x - 1)}{2\sqrt{x}} \)

Setting \( y' = 0 \Rightarrow x = 1 \).

Point: (1, -2). (Note: \( x=0 \) is a boundary point).

4. Inflection Points

Second derivative:
\( y'' = \frac{3}{4}x^{-1/2} + \frac{3}{4}x^{-3/2} = \frac{3(x + 1)}{4x^{3/2}} \)

Setting \( y'' = 0 \Rightarrow x = -1 \). (Not in domain \( [0, \infty) \)).
None.
